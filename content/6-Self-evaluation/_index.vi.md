### Tổng Quan Thực Tập

Trong suốt thời gian thực tập tại **Amazon Web Services (AWS)** từ **08/09 đến 09/12/2024**, tôi đã có cơ hội quý báu để học hỏi, rèn luyện và áp dụng kiến thức đã được trang bị tại trường vào môi trường điện toán đám mây thực tế.

Tôi đã tham gia xây dựng **Nền tảng Học IELTS Bandup** - một ứng dụng AI toàn diện giúp học sinh chuẩn bị cho kỳ thi IELTS thông qua đánh giá Writing và Speaking tự động, cũng như tạo Flashcard thông minh từ tài liệu học tập.

---

### Thành Tựu Chính

Qua dự án này, tôi đã phát triển và cải thiện các kỹ năng:

**Kiến Trúc Cloud & Dịch Vụ AWS:**
- Thiết kế và triển khai kiến trúc serverless có khả năng mở rộng sử dụng AWS Lambda, API Gateway và SQS
- Cấu hình mạng VPC với public/private subnets, NAT Gateway và Security Groups
- Thiết lập điều phối container với Amazon ECS và Fargate
- Quản lý lưu trữ dữ liệu với Amazon S3, DynamoDB và ElastiCache (Redis)
- Tích hợp dịch vụ AI bao gồm Amazon Bedrock (Titan Embeddings) và Google Gemini API

**Phát Triển & DevOps:**
- Xây dựng Lambda functions bằng Python cho đánh giá AI
- Triển khai pipeline RAG (Retrieval-Augmented Generation) cho tạo flashcard
- Thiết lập CI/CD pipelines với GitLab và AWS CodePipeline
- Thực hành nguyên tắc Infrastructure as Code

**Tích Hợp AI/ML:**
- Tích hợp Google Gemini API cho xử lý audio gốc (tiết kiệm 72% chi phí so với phương pháp truyền thống)
- Triển khai Amazon Titan Text Embeddings V2 cho tìm kiếm vector similarity
- Thiết kế chiến lược prompt engineering cho chấm điểm band IELTS
